# Glossary

- **Batch:** A single production run (e.g., a kettle run) that yields sellable units.
- **Sellable Unit:** The unit sold to customers (e.g., one bag or cup).
- **Reconciliation:** End-of-day process of validating sales totals, inventory consumption, and cash/card records.
- **Par Level:** Minimum on-hand quantity before reordering is required.
- **Adjustment:** A manual change to inventory to account for spoilage, loss, donation, or corrections.

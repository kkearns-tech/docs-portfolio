# Prep Day Workflow

## Goal
Prepare ingredients, packaging, and equipment so event day runs smoothly.

## Inventory check
- [ ] Review on-hand inventory against par levels
- [ ] Create a restock list (ingredients + packaging)
- [ ] Confirm equipment readiness (kettle, power, tables, signage)

## Prep tasks
- [ ] Stage ingredients by expected consumption
- [ ] Pre-label packaging if used
- [ ] Prepare limeade station materials (if applicable)
- [ ] Prepare cleaning supplies and waste plan

## Load-out checklist
- [ ] Ingredients
- [ ] Packaging (bags/cups/lids/napkins)
- [ ] Payment supplies (square reader/backup battery/receipt plan)
- [ ] Signage and menu
- [ ] Tools, gloves, cleaning materials

## Outputs
- Restock plan
- Load-out list for event
- Notes for improvements

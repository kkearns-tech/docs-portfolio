# Product Overview

## Purpose
Popstiche Ops is a lightweight operational documentation set and system blueprint for running event-based sales efficiently and consistently.

## Primary goals
- Reduce event-day operational overhead
- Improve inventory visibility and replenishment accuracy
- Make end-of-day reconciliation repeatable
- Establish a clear shared language (terms, roles, workflows)

## Non-goals (v0)
- Replacing point-of-sale systems
- Advanced forecasting
- Multi-location warehousing complexity
